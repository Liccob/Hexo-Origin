最简单的闭包面试题
===

 
```
var data = [];
for(var i = 0 ; i<3 ;i++){
  data[i]= (function(i){
    return function(){
      console.log(i)
    }
  })(i)
}
data[0]();//0

//我们从函数执行上下文来分析
data[0]'s context = {
  scope=[本函数.AO, 匿名函数context.AO, globalContext.VO]
}

//函数执行时，其实是从本函数的作用域链开始找，找不到就往上一个作用域找，本函数找不到，则往匿名函数的AO寻找，匿名函数由于是立即执行函数，则在赋值给data[0]时，传入的实////参i为0，则其AO则如下；
匿名函数context = {
  AO:{
    arguments: {
      0: 0,
      length: 1
    },
    i: 0
  }
}
```
这个时候就找到 i是0
